Projet : produire un document avec une IA générative

    Dans cette page

    • Protocole

    Rédiger un prompt adapté qui permet de (au choix) :

    • - créer une affiche de prévention pour des collégiens
    • - créer un quiz de révision
    • - créer une mascotte pour l’Atelier Codage
    • - faire une fiche de révision pour le brevet
    • - une autre idée au choix (à faire valider par le prof)

    Protocole

    • Faire la liste des données utiles à l’IA.
    • Faire la liste des données à NE PAS lui fournir.
    • Choisir une méthode de prompt adaptée.
    • Répartir les éléments dans le prompt.
    • Tester votre prompt sur Duck.ai.
    • Modifier au moins une fois le prompt après avoir obtenu la première réponse.
    • Imprimer votre prompt final et la réponse fournie par l’IA.
    • Vérifier le contenu avec l’aide du cours, d’un document de référence ou d’un professeur.
